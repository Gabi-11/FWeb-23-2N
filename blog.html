Meu nome é Gabriel de Graaf, tenho 18 anos, estou fazendo o curso técnico de desenvolvimento de sistema na sede do SENAC, dentro da Unisinos, e esse é o meu blog. Dentro dele eu vou colocar meus principais trabalhos, ou o que eu achei mais interessante de realizar sozinho, junto ao professor ou com meus colegas do curso, e é isso, esses são os principais trabalhos que eu fiz no período de curso do SENAC usando o HTML.

Este projeto foi desenvolvido em conjunto com meu professor e consiste em uma Pokédex interativa. Nela, é possível visualizar informações detalhadas sobre diferentes Pokémon, como nome, imagem, peso, altura e tipagem. O design foi pensado para lembrar a interface clássica da Pokédex dos jogos, proporcionando uma experiência nostálgica e funcional.

A calculadora foi criada com HTML, CSS e JavaScript, permitindo realizar operações matemáticas básicas de forma prática. O layout é intuitivo, com botões para números e operações, além de um visor para exibir os cálculos em tempo real. Esse projeto foi uma ótima oportunidade para trabalhar com manipulação de eventos e lógica de programação.

Este guia oferece um passo a passo detalhado sobre como utilizar JavaScript.
